# 03-calcolatrice-java
Repository dedicato allo sviluppo progressivo di una calcolatrice in Java, realizzato con finalità didattiche.

# Progetti da console
**:one: Calcolatrice base:**
Implementazione delle operazioni aritmetiche fondamentali, utilizzo della classe *Scanner* per l'input utente, ciclo *do-while*; costrutto *switch-case* per la gestione del menu e strutture condizionali *if-else*.

**:two: Calcolatrice intermedia:**
Estensione delle funzionalità tramite l'uso della classe Math, con l'aggiunta di operazioni come la radice quadrata e l'elevamento a potenza.

**:three: Calcolatrice Avanzata:**
Versione più complessa, caratterizzata dall'utilizzo di costrutti e cicli annidati e dall'implementazione del simbolo **=** per confermare l'operazione e restituire il risultato all'utente, simulando il comportamento di una calcolatrice reale.

**:four 4️⃣: Calcolatrice Web - Java Spring Boot & ANgular**
Progetto didattico full stack che evolve una calcolatrice da applicazione console web application
