# Calcolatrice Web – Java Spring Boot & Angular

Progetto didattico full stack che evolve una calcolatrice da applicazione console a web application.

## Tecnologie utilizzate
- Java 17
- Spring Boot (REST API)
- Angular 16
- Bootstrap 5
- Maven

## Funzionalità
- Operazioni aritmetiche base
- Elevamento a potenza
- Radice quadrata
- Gestione errori (divisione per zero, input non validi)
- Frontend Angular collegato a backend Spring tramite REST

## Architettura
- Frontend: Angular + Bootstrap
- Backend: Spring Boot REST API
- Comunicazione: HTTP GET (JSON)
